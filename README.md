## Environment setup

This project now reads Firebase and notification credentials from a `.env` file.

1. Copy `.env.example` to `.env`.
2. Fill all values in `.env`.
3. Run `flutter pub get`.
4. Start the app.

If any required variable is missing, startup or notification auth will throw a clear error.
